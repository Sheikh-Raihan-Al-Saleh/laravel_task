1. git clone https://github.com/Sheikh-Raihan-Al-Saleh/laravel_task
2. Run composer update  or composer install
3. Run this command to your terminal  cp .env.example .env
4. Run php artisan key:generate
5. Run php artisan migrate
6. Run php artisan db:seed --class=UserSeeder
7. Run php artisan db:seed --class=ProductSeeder
8. Finally Run the project using php artisan serve
